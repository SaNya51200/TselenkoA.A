# Лабораторная работа 3: JavaScript и современный фронтенд

## Описание
Изучение применения функционального программирования в JavaScript.

## Структура проекта
- `array-methods.js` - методы массивов (map, filter, reduce)
- `functions-closures.js` - стрелочные функции, замыкания, каррирование
- `immutability.js` - иммутабельные обновления
- `async-fp.js` - асинхронное функциональное программирование
- `react-functional.js` - функциональные компоненты React и хуки
- `index.html` - демонстрационная страница

## Запуск

### В браузере
Откройте `index.html` в браузере и используйте консоль разработчика (F12).

### Через Node.js
```bash
node array-methods.js
node functions-closures.js
node immutability.js
```

### React компоненты
Для работы с React компонентами требуется установка:
```bash
npm install react react-dom
```
