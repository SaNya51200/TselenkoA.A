# Лабораторная работа 2: Python и функциональные возможности

## Описание
Изучение возможностей функционального программирования в Python.

## Структура проекта
- `functions_as_objects.py` - функции как объекты первого класса
- `lambda_closures.py` - lambda-функции и замыкания
- `higher_order.py` - функции высшего порядка (map, filter, reduce)
- `comprehensions_generators.py` - списковые включения и генераторы
- `decorators.py` - декораторы
- `main.py` - главный файл с демонстрацией

## Запуск
```bash
python main.py
```

Или отдельные модули:
```bash
python functions_as_objects.py
python lambda_closures.py
python higher_order.py
python comprehensions_generators.py
python decorators.py
```
